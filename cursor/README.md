# Cursor marketplace

Assets and submission tracking for the Cursor MCP directory.

## Install snippet

```json
{
  "mcpServers": {
    "1inch-business": {
      "url": "https://api.1inch.com/mcp/protocol"
    }
  }
}
```

Save as `.cursor/mcp.json` in your project root.

# Claude Connector

The 1inch MCP server is available as a connector in the [Claude Connectors Directory](https://claude.com/connectors).

**Server URL:** `https://api.1inch.com/mcp/protocol`

Search for **1inch** in Claude's connector settings, or see the [setup guide](https://business.1inch.com/portal/documentation/ai-integration/supported-clients) for manual configuration.

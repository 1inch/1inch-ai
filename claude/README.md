# Claude Connectors Directory

Assets and submission tracking for the Anthropic Connectors Directory.

## Server details

- **URL:** `https://api.1inch.com/mcp/protocol`
- **Transport:** Streamable HTTP
- **Auth:** OAuth 2.0 + API key (`Authorization: Bearer`)
- **Tool annotations:** All tools have `readOnlyHint` or `destructiveHint`
